# Online-Food-Delivery-Customer-Analytics
End-to-End Data Analyst Project using Python, SQL, Excel, and Power BI

## Project Overview
This project analyzes online food delivery data to understand customer behavior, revenue trends, restaurant performance, and delivery efficiency.

## Tools Used
Python
SQL
Excel
Power BI

## Project Workflow
1. Generated 500 rows dataset using Python
2. Performed Exploratory Data Analysis (EDA)
3. Created Pivot Tables in Excel
4. Imported dataset into MySQL
5. Wrote SQL queries for analysis
6. Built interactive Power BI dashboard

## Key Insights
• Mumbai generated the highest revenue
• Indian cuisine is the most popular
• UPI is the most used payment method
• Spice Hub is the top-performing restaurant
• Most orders were delivered on time
